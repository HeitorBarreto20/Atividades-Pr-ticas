Listas

Para colocar o tipo de uma lista ordenada de onde começa se usa menor que "ol" "type=" "start=" maior que. Exemplo:

  1. Listas
  2. Senac
  3. ZN

Para colocar o tipo de lista em uma lista não ordenada se usa menor que "ul" "type=" maior que. Exemplo:

  • Listas
  • Senac
  • ZN

Listas Ordenadas se usa menor que "ol" maior que. Exemplo:

  1. Senac
  2. ZN
  3. RN

Listas Não Ordenadas se usa menor que "ul" maior que. Exemplo:

  • Senac
  • ZN
  • RN

Listas de Definição usando "dl", depois dentro da lista usa "dt" e a definição é "dd". Exemplo:

Senac

Links

Você pode acessar o meu link do GitHub
